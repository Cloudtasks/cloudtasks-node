> Cloudtasks.io nodeJS SDK


## Install

```sh
$ npm install --save cloudtasks-node
```


## Usage

```js
var cloudtasks-node = require('cloudtasks-node');

cloudtasks-node('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
